Let me think through what this property actually is before proposing where it goes.

What "Documentation Level" actually is — two registers
It operates simultaneously as source metadata and as site property:
Register 1 — Source tier (what kind of document did we receive?)
This affects evidentiary weight across all stages. A nomination dossier, an excavation report, and a heritage inventory entry all describe a site, but they're epistemically different objects. The source tier tells you: what was this document designed to do, what biases does that purpose introduce, and how far is it from primary data?
Register 2 — Documentary Value (how well-documented is the site itself, regardless of what we received?)
This is an attribute of the site — it feeds into [CA-V] Documentary/Archival Value and into Stage 3 integrity. A site can have rich Tier 1 excavation archives (primary field records) that constitute genuine heritage value, even if all we received is a Tier 3 nomination dossier.
These are related but not the same. Tabgha illustrates both: the source is Tier 3 (nomination dossier), but the site has decades of excavation records (Schneider, Loffreda, Bagatti) that are probably Tier 1 — those records ARE documentary value, and we currently can't see them.
