# Build Your Own AI Assistant

An AI that helps you design your own AI agent — choose its expertise, principles, and workflow, then deploy it.

## Quick Start

[Open in Gemini](https://gemini.google.com/gem/1LbC3oHGIS83rP8uWdIEEeaU9_ixfEMh1?usp=sharing) — click and start building. No setup needed.

## Deploy What You Built

After the bot generates your system instructions, copy the text and paste it into:

- **Claude** — [claude.ai](https://claude.ai) → Projects → New Project → Custom Instructions
- **ChatGPT** — [chatgpt.com](https://chatgpt.com) → Explore GPTs → Create → Instructions field
- **Gemini** — [gemini.google.com](https://gemini.google.com) → Gem Manager → New Gem → System Instructions

Then give it a task and see how it responds. The file is yours to keep refining.

## Manual Setup

If you prefer to set it up yourself, copy [`agent-for-agents.md`](agent-for-agents.md) and paste it as system instructions on any platform.
